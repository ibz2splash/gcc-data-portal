# GCC Data Reuse Portal

An interactive Streamlit-based analytics platform demonstrating the reuse of GCC statistical data across four policy domains:
- Tourism Futures
- Smart City Readiness
- Youth Opportunity Radar
- Trade & Logistics Explorer

## How to Run
1. Install Python 3.10+
2. Install dependencies:
   pip install -r requirements.txt
3. Launch the app:
   streamlit run Home.py

## Data Sources
All datasets are derived from publicly available GCC statistical portals and processed into harmonized indicator-level CSV files.

## Key Features
- Cross-country benchmarking
- Indicator filtering and time-series visualization
- Scenario-ready architecture
- Downloadable filtered datasets
- Unified UI styling across modules

## Project Structure
- Home.py: Main landing page and navigation
- pages/: Individual platform modules
- data/processed/: Cleaned and harmonized datasets
- assets/: Visual assets
- src/ui_style.py: Global UI styling
